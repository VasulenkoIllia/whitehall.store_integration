# Поточний функціонал (простими словами)

## Дані постачальників
- Можна додавати постачальників, вмикати/вимикати, змінювати пріоритет і націнку.
- Для кожного постачальника можна додавати кілька джерел Google Sheets.
- Мапінг колонок робиться окремо для кожного джерела.

## Імпорт і підготовка
- Імпорт читає всі джерела з таблиць, обробляє рядки та пише у базу.
- Порожні/некоректні рядки пропускаються з логуванням причин.
- Можна робити імпорт по одному постачальнику або по всіх.

## Finalize (фінальні дані)
- Націнка по постачальнику.
- Мінімальна націнка (увімкнено/вимкнено).
- Округлення до 10 грн.
- Дедуплікація по `article + size` (з урахуванням пріоритету/ціни).
- Можна встановити ручну ціну для конкретного товару.

## Horoshop
- Є синхронізація каталогу Horoshop (дзеркало).
- Є прев’ю API‑даних, які підуть у Horoshop.
- Є експорт у файл (manual download) для Finalize та Horoshop preview.
- Є ручне відправлення даних у Horoshop API.

## Планування
- Є плановий процес “імпорт → finalize → export → Horoshop”.
- Можна вибрати час запуску та постачальника для Horoshop.
- Статус і результати процесу видно на головній.

## Логи та контроль
- Логи відображаються в адмінці.
- Помилки можуть дублюватися у Telegram.
- Є можливість скасувати job або дочекатися таймауту.

## Режими доступу
- `/admin` — повний доступ.
- `/` — тільки перегляд (без змін).
